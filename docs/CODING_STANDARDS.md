# Coding Standards – AAI

## Backend (FastAPI)

- Follow PEP8
- Use type hints
- Use dependency injection
- No business logic in route functions
- Use response_model in endpoints
- Validate inputs via Pydantic schemas
- No direct raw SQL

---

## Frontend

- TypeScript strict mode
- Functional components only
- No inline scripts
- Separation of UI and logic

---

## Git Practices

- Meaningful commit messages
- Small pull requests
- No direct commits to main
- PR must pass CI checks
