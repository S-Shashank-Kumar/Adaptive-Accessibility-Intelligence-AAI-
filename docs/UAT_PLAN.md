# User Acceptance Testing Plan – AAI

## 1. Objective

Validate that the MVP functions correctly in UAT environment.

---

## 2. Test Cases

| ID | Scenario | Expected Result |
|----|----------|----------------|
| UAT-01 | Paste complex text | Simplified text returned |
| UAT-02 | Enable high contrast | UI updates immediately |
| UAT-03 | Click read aloud | Speech plays |
| UAT-04 | Dictate text | Text appears correctly |
| UAT-05 | Invalid login | Error message displayed |

---

## 3. Acceptance Criteria

- All core features functional
- No critical security issues
- No API crashes
- Performance < 5 seconds response
